Krótko o programie

Visual Studio Code – darmowy edytor kodu źródłowego z kolorowaniem składni dla wielu języków, stworzony przez Microsoft, o otwartym kodzie źródłowym.

Skróty klawiszowe

CTRL + Z cofanie wykonanej akcji
CTRL + Y przywrócenie wykonanej akcji
ALT + F4 większa wydajność programu
CTRL + A zaznaczenie całego dokumentu
